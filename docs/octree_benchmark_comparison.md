# Octree KNN ベンチマーク比較

Octree の近傍候補管理を最大ヒープ方式で実装した最新版と KDTree を比較した結果です。ベンチマークテスト `OctreeTest.BenchmarkOctreeAgainstKDTree` をウォームアップ後に 100 回平均で計測し、点群構成はテストコードのデフォルト設定に従っています。
